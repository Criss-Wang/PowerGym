HERON Core Framework

HERON (Hierarchical Environments for Realistic Observability in Networks) is a domain-agnostic
multi-agent reinforcement learning framework.

Modules
-------

- :doc:`agents` - Hierarchical agent abstractions (Field, Coordinator, System, Proxy)
- :doc:`core` - Core data structures (Action, Observation, State, Feature)
- :doc:`protocols` - Coordination protocols (Vertical, Horizontal)
- :doc:`messaging` - Message broker system
- :doc:`envs` - Base environment interfaces
